📊 Data Quality Pipeline using Python + MySQL
 Project Overview

This project is a Data Quality Analysis System built using Python and MySQL.
It connects to a MySQL database, analyzes a dataset, detects data issues, and generates a data quality report.

The system identifies:

Missing (NULL) values
Duplicate records
Invalid data (negative or incorrect values)
Clean dataset extraction
Overall data quality score
 Problem Statement

Real-world datasets often contain:

Missing values
Duplicate records
Incorrect or invalid entries

This project solves these problems by automatically analyzing data quality and producing a structured report.
